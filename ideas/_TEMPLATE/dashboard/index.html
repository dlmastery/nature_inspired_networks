Where the inputs come from

For each idea, the renderer reads ideas/<NN>/hillclimb_results.json (the summary) plus each cell's experiments/cifar100/<tag>__hc_<config_id>_seed<S>/metrics.json file. The 5-section page is regenerated by running: 

python ideas/_TEMPLATE/dashboard/render.py            # all three winners
python ideas/_TEMPLATE/dashboard/render.py --idea 09_phi_budget  # just one
